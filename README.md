# Robotics-Lab
Contiene tutti gli homework svolti per il corso di Robotics Lab. L'obiettivo era quello di sviluppare un controllo per il KUKA IIWA. Come software: Gazebo e RViz sono stati usati per il carimento del modello del robot e dell'ambiente; ROS per definire i vari nodi e programmi per caricare mondo e struttura cinematica del robot; KDL è la libreria usata per calcolare e definire la struttura cinematica.
