# Robotics-Lab
The aim of the exam was to develop a dynamic control system for the KUKA IIWA, using the following software tools:
- ROS: to define the nodes and programs required to load the world, the robot, and manage the control system;
- Gazebo and RViz: to visualize and simulate the robot's environment;
- KDL (Kinematics and Dynamics Library): to define the robot's kinematic and dynamic structure.
